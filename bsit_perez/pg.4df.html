What is data format?

Data Formats

CSPT

Everything in the digital world is built on digits. Every method does calculations on numbers, and most critically, a computer can only interpret binary (digits 0 & 1). The system will use these numbers as input data to fetch, process, and carry out the instruction. Data is the unprocessed information or group of instructions that are given to the system for execution in order to produce the desired output. Data formatting is the process of entering the data into the system in a certain format that meets the system's requirements.

Data Formats

  • The organization and structure of information recorded in a computer system is known as a data format. In order for data to be understood, processed, and shared by various computer systems and applications, it must be encoded and represented in a certain way, which is specified by a data format.
  • Issues around data formats

    • To ensure sharing, ongoing access, and data preservation, file formats should be adopted. Choose reusable, open formats and open standards. The features that might be lost when the files are converted to their sharing and preservation format, as well as any particular software that will be required to view or work with the data, should be mentioned in your documentation if you are using a different format during the collection and analysis phases of your research.

  • Recommended Digital Data Formats

    • Text, Documentation, Scripts: XML, PDF/A, HTML, Plain Text. Still Image: TIFF, JPEG 2000, PNG, JPEG/JFIF, DNG (digital negative), BMP, GIF.

    Geospatial: Shapefile (SHP, DBF, SHX), GeoTIFF, NetCDF.

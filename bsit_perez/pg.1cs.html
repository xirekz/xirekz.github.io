What is computer system?

Computer System

CSPT

Computer systems are the collection of hardware and software elements that cooperate to carry out various computing operations. A central processing unit (CPU), memory, input/output devices, and storage devices are the basic components of a computer system. The operating system, programs, and utilities that govern and operate the computer system are included in the software components. The underlying hardware and software architecture on which a specific computing system is constructed is referred to as the platform. A platform offers a collection of tools and services that allow programmers to build and distribute software applications. A platform might be software-based, like an operating system or cloud computing environment, or hardware-based, like a computer system or mobile device.
